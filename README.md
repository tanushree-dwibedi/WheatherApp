# 🌦️ Weather App

A simple and responsive weather application that fetches real-time weather data using an API.

## 🚀 Features
- Search weather by city name
- Displays temperature, humidity, and wind speed
- Real-time API data fetching
- Error handling for invalid city names
- Clean and responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Weather API (e.g., OpenWeatherMap)

## 🔗 How it Works
- User enters a city name
- App sends a request to the weather API
- Receives data in JSON format
- Updates UI with temperature, humidity, and wind details
